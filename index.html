Rutas Seguras Kids es una aplicación frontend desarrollada con JavaScript Vanilla, HTML y CSS que gestiona de forma dinámica la logística de rutas escolares y estudiantes. Destaca por el uso de Web Components nativos, manipulación avanzada del DOM y el consumo asíncrono de APIs externas para mostrar alertas climáticas en tiempo real, todo construido sin frameworks para garantizar un código limpio y eficiente.
